# 시가 총액 순위 페이지 — Title & Description 메타 템플릿

타깃 키워드별 SEO 메타데이터 모음입니다. 각 페이지의 `<title>`과 `<meta name="description">`에 그대로 적용할 수 있습니다.

---

## 템플릿 구조

**Title 템플릿**

```
{시장} 시가 총액 순위 TOP 100 | {보조문구}
```

**Description 템플릿**

```
{시장} 시가 총액 순위 상위 100개 기업의 기업 가치를 확인하세요. {종목1}, {종목2}, {종목3}, {종목4}, {종목5} 등 시총 상위 기업의 순위와 주가, 등락률을 빠르게 비교할 수 있습니다.
```

치환 변수: `{시장}`, `{보조문구}`, `{종목1}`~`{종목5}`

## 적용 참고사항

- **대표 종목 동적 치환 권장**: `{종목1}`~`{종목5}`를 각 페이지의 실제 시총 1~5위 데이터와 연동하면 순위 변동이 자동 반영되고, 페이지 간 문장도 자연스럽게 달라져 중복 콘텐츠(duplicate content) 우려를 줄일 수 있습니다.
- **종목명 기준**: 위 종목은 작성 시점 기준 시총 상위 기업입니다.
- **Title 길이**: 모두 공백 포함 30자 안팎으로 구글 제목 노출 한도(약 30~35자) 이내입니다.
- **Description 길이**: 공백 포함 약 90~100자로 구글 데스크톱 노출 한도(약 155자) 이내입니다.
- **보조문구 대안**: 파이프(|) 뒤 보조문구를 사이트 브랜드명으로 통일하면 브랜드 인지도에 유리하고, 키워드형(KOSPI·NASDAQ 등)으로 두면 영문 검색 유입에 유리합니다.
